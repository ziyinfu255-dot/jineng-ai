# jineng-ai

Codex skills collected for reuse.

## Included skills

- `class-meeting-packager`
- `find-skills`
- `course-cover-images`

## Usage

Copy a skill folder from `skills/` into your Codex skills directory, for example:

```powershell
Copy-Item -Recurse .\skills\find-skills $env:USERPROFILE\.codex\skills\
```
